# /assets/sketches

Croquis SVG illustrant les traits caractéristiques pour la validation guidée.

## Convention de nommage

- Un sous-dossier par espèce : `{species-id}/`
- Un fichier SVG par trait : `{trait-id}.svg`
- Exemple : `ulmus-minor/leaf-asymmetric-base.svg`

## Critères techniques

- Format : SVG (vectoriel)
- Optimisé via SVGO ou équivalent
- Cible : < 10 ko par fichier
- ViewBox par défaut : `0 0 400 300` (à adapter selon le trait)
- Contient les éléments `<title>` et `<desc>` pour l'accessibilité

## Convention stylistique

À figer en phase 1 sur les espèces pilotes. À documenter ici une fois définie.

Voir `/docs/06-sketches-guide.md` pour le guide complet de production.

## État de couverture

À compléter au fur et à mesure de la production des croquis.
